# 💼 Portafolio — German Candelario Hernández

Portafolio profesional desarrollado con **React** y **Vite**, desplegado en **Vercel**.

## 🚀 Stack Tecnológico

- ⚛️ React 18
- ⚡ Vite 5
- 🎨 Bootstrap 5 + Bootstrap Icons
- 🖋️ Google Fonts (Outfit)
- 🎭 Vanilla CSS con Glassmorphism

## 📦 Ejecución Local

```bash
npm install
npm run dev
```

El servidor se levanta en [http://localhost:5173](http://localhost:5173)

## 🏗️ Build de Producción

```bash
npm run build
```
